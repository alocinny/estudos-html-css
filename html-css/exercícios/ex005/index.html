Formatação de texto

Endereço

Para adicionar um endereço, você deve usar a tag <address>
Exemplo:
Juvenal era um sujeito de muita sorte. E já começou de pequeno onde morou na

Rua Marquês de Lira filho.

Negrito e Itálico

Para colocar em negrito, deve usar a tag <strong>

para oclocar em itálico, deve usar a tag <em>

Marca Texto

Para marcar uma palavra, deve usar a tag <nark>

Small

Para deixar a palavra pequena, deve usar a tag <small>

Texto deletado

Para deletar um texto, deve usar atag <del>

Texto inserido

Para inserir um texto, deve usar a tag <ins>

Texto sobrescrito e subscrito

para colocar um texto sobrescrito, deve usar a tag <sup>

Para colocar um texto subscrito, deve usar a tag <sub>

Trechos em código

para colocar um texto em código, deve usar a tag <code<

Para manter o código pré-formatado, deve usar a tag <pre>

Citações

Citações simples

Para fazer uma citação simples, deve usar a tag <q>

Citação em bloco

Para criar citações em blocos, deve usar a tag <blockquote>

Abreviações

Para adicionar abreviações, você deve usar a tag <abbr>

Texto Invertido

Para inverter um texto, deve usar a tag <bdo<, colocando "rtl" ou "ltr" como valor do parâmetro 'dir=""'
