Parágrafo

Adicionar um parágrafo

Para adicionar um parágrafo, você deve usar a tag <p>

Quebrar um parágrafo

Para quebrar um texto em algumn lugar específico, você deve usar a tag <br>

Caracteres

Colchetes angulares

para adicionar colchetes angulares deve colocar:

  • Um & l t; para colocar o símbolo menor que
  • um & g t; para colocar o símbolo maior que

Existem outros símbolos que podem ser exibidos usando códigos. Esses códigos são chamados de HTML Entities.

Emojis

para adicionar emojis você deve copiar o código do emoji escolhido e substituir o "U+" por "&#x"
